# 메모 웹 애플리케이션

Notion과 iCloud Notes에서 영감을 받아 만든 메모 앱입니다.

## 주요 기능
- Tiptap 기반 에디터
- 체크리스트, 텍스트 꾸미기
- 표 삽입 및 우클릭 메뉴
- 반응형 디자인

## 사용 기술
- React, Next.js, TypeScript
- Tailwind CSS, Tiptap, DnD Kit
